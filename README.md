# 專案名稱：基於毫米波雷達之室內人流追蹤系統

## 1. Introduction (研究導論)

*   **A. Attention Getter:** 隨著智慧建築與物聯網（AIoT）的普及，精準掌握場域人數已成為節能管控、空間優化及公共安全的核心需求。
*   **B. But/However:** 現有技術存在顯著缺陷：影像辨識方案（如 YOLO）涉及嚴重的隱私爭議且受光照影響；傳統紅外線（PIR）則無法偵測靜止人體，且在多人同時進出時精準度大幅下降。
*   **C. Cure:** 本研究提出一種基於毫米波雷達 (mmWave Radar) 的解決方案。利用毫米波對位移與微動的高靈敏度，在完全不侵犯個資隱私的前提下，達成全天候的人流監測。
*   **D. Development:** 系統設計基於 FMCW（調頻連續波）技術與有限狀態機（Finite State Machine）演算法。透過定義虛擬的進入與離開偵測區，結合 ESP32 進行邊緣運算，即時判斷人員流動方向。
*   **E. Experiments:** 本專案在標準教室門口進行多種場景測試，包含單人快速進出、多人併行、以及靜止站立於偵測區等壓力測試，以評估系統的穩定性與計數準確度。
*   **F. Findings:** 實驗結果顯示，本系統能有效克服光照與隱私問題，並在多人進出時透過距離門檻過濾大幅降低誤判率，提供了一種低功耗且高可靠性的空間管理方案。

## 2. Related Work (相關研究與技術回顧)

**NotebookLM 輔助摘要：**

*   **mmWave vs. PIR:** 毫米波雷達可透過多普勒頻移偵測到公釐等級的微動（如呼吸），解決了 PIR 僅能感應大幅度移動的痛點。
*   **Privacy-First Design:** 參考現行 GDPR 規範，影像計數系統在公共空間佈署困難度增加，毫米波數據僅回傳距離、速度與能量值，本質上具備匿名性。
*   **FMCW 原理：** 藉由發射頻率隨時間線性變化的連續波，計算發射訊號與接收訊號的頻率差，進而精確測距。

## 3. Proposed Design (系統設計與規格)

### A. 系統架構圖
系統分為「感知層」、「處理層」與「應用層」。

### B. 硬體規格 (Specifications)
*   **感測器：** HLK-LD2410 毫米波雷達模組（具備靜止/運動雙模式偵測）。
*   **主控板：** ESP32 微處理器（負責 UART 數據讀取與 Wi-Fi 通訊）。
*   **電源：** 5V Micro USB / 行動電源供電。

### C. 軟體規格
*   **開發語言：** C++ (Arduino IDE) / Python (後端分析)。
*   **通訊協議：** HTTP Post 傳輸至網頁。
